Final Project Proposal

MatterStick

Project designed and produced by Steven Roberts

The user will be able to calculate the answers to simple physics and chemistry problems

Will use python functions to calculate answers to problems based on the user's needs

Last updated 12/4/22

Features include:

Calculating the velocity final of a mass

Recognize what type of problem is to be solved based on the variables inputted

Add a simple drawing of what the problem looks like

Future Features:

Create a graph of a component vs. time or distance

Use different colors in the graph for different variables

Originally MatterStick was going to be able to identify what type of problem was to be solved based of the variables. I decided to change that when I realized it would be better to have the user select what type of problem they want to have solved. Originally I also wanted there to be graphs and images representing visually what each problem looks like, however that proved to be more challenging than anticipated so I used the graph to represent a sample problem for problem 3, where an object is thrown off of a cliff. If there were more time I would have made a graph to represent each, specific calculation that user inputted. I also would have included different colors for each of these graphs so that they are unique to each problem. Additionally, in the future I would add several different types of more complex problems, such as using calculus to solve these types of problems and including more challenging problems.
